== Description ==

This simple plugin will grab the NASA Astrology Picture of the Day and create a new post with it. You can choose which username it will post under as well as whether or not to go ahead and publish on creation. 

Using this plugin will required you get an API Key from NASA, which is very easy to get.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the entire `nasa-potd` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Update the API Key, default post status and default poster settings. 

== Frequently Asked Questions ==

= Is this fully automated? =

Yes, this plugin is fully automated, once you set it up it will grab the Astrology Picture of the Day, once a day and create a post with it.

= Where do I get an API Key? =

This is very simple, just go to the NASA website and apply for one. https://data.nasa.gov/developer/external/planetary/#apply-for-an-api-key
